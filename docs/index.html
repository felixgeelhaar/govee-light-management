What's inside · v2.7

Five hybrid actions,
plus the tools you reach for

Each hybrid action works on both Stream Deck keys and Stream Deck+ dials. Drop once, use anywhere.

Recall New

One key, any saved look. Picks from every Govee dynamic scene, your DIY moods, and snapshots — in a single dropdown.

On / Off

Toggle a light or a whole group. Three-state title (●/◐/○) with N/M count for groups — never lies about mixed state.

Brightness

Press for a fixed %. Drop the same action on a Stream Deck+ dial: rotate to adjust, press to toggle power.

Color

Press for a fixed RGB pick. Or rotate the dial through the full hue wheel with live colour feedback on the LCD.

Color Temperature

Press for a fixed Kelvin. Rotate the dial between 2000K warm candlelight and 9000K crisp daylight on a gradient bar.

Segment Color

For RGB IC strips: press for rainbow / solid / gradient presets across a range. Or rotate the dial to tune one segment at a time.

Saturation

Press for a fixed %. Rotate to dial through grey-to-vivid on the LCD. Same hybrid pattern as the rest.

Music Mode + Snapshots

Audio-reactive lighting, saved Govee presets, feature toggles (Nightlight, Gradient, DreamView), Schedule, Sequence, Custom Effect.

Honest group state

Group apply succeeds for some and fails for others? You see "⚠ N/M failed" for 30s. No more silent green checkmarks.

How it works

Set up in under a minute

No accounts to create, no servers, no configuration files. Just paste an API key and go.

Install the plugin

Download from the Elgato Marketplace. The plugin appears instantly in your Stream Deck app.

Get your Govee API key

Open the Govee app, tap your profile, and request a free developer API key. It arrives by email.

Drag and configure

Drop an action on a key or a Stream Deck+ dial — same action, same settings. Press for a fixed value, rotate to adjust.

Ready to light up your setup?

Free, open source, and built to stay out of your way.
