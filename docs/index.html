What's inside

Everything you need
for tactile light control

Nine purpose-built actions for keypads and dials. No phone, no lag, no fuss.

On / Off

Toggle individual lights or whole groups. The action remembers its state across sessions.

Brightness

Tap a preset or dial it in continuously. Stream Deck+ dials feel like a real hardware knob.

Color

Scrub the full hue wheel with live feedback on the touchscreen. Snap to any color in under a second.

Color Temperature

Shift between 2000K warm candlelight and 9000K crisp daylight with a gradient dial.

Segment Control

For RGB IC strips: apply rainbow, solid, or gradient presets across every segment — or tune them individually.

Stream Deck+ Dials

Custom touchscreen layouts with branded gradient bars and bold values. Five purpose-built encoder actions.

Fast & Resilient

Built on an enterprise-grade API client with rate limiting, retry, and circuit-breaker protection.

Private by Design

Your API key stays on your machine in Stream Deck's encrypted storage. No analytics, no tracking, ever.

Groups

Control multiple lights as one with saved groups. Build whole-room scenes with a single press.

How it works

Set up in under a minute

No accounts to create, no servers, no configuration files. Just paste an API key and go.

Install the plugin

Download from the Elgato Marketplace. The plugin appears instantly in your Stream Deck app.

Get your Govee API key

Open the Govee app, tap your profile, and request a free developer API key. It arrives by email.

Drag and configure

Drag any action onto a button, paste the key once, and pick a light. You're done.
