How it works

Set up in under one minute

No accounts. No servers. No configuration files.

01

Install

Download the plugin from the Elgato Marketplace. It appears instantly in your Stream Deck app.

02

Get your API key

Request a free Govee developer API key from the Govee app. It arrives by email in seconds.

03

Drag and go

Drop an action on a key or a Stream Deck+ dial — same action, same settings. Press for a fixed value, rotate to adjust.
